# Installation:
```
git clone https://github.com/hiddify/hiddifypanel_bot

cd hiddifypanel_bot
cp .env_example .env
#Enter the user panel link and admin panel link and bot token in .env file
nano .env 
pip install -e .
hiddifypanel_bot
```
